# 🎭 Sistem Casting Model

Sistem Casting Model adalah aplikasi berbasis C++ yang dirancang untuk membantu panitia mengelola data peserta casting model secara cepat dan terorganisir.

Mulai dari pendaftaran peserta, pencarian data, pengurutan berdasarkan berbagai kriteria, hingga analisis BMI dan statistik peserta, semuanya dapat dilakukan melalui satu program sederhana berbasis terminal.

---

## ✨ Apa yang Bisa Dilakukan Program Ini?

### 👤 Menambah Peserta Baru
Masukkan data peserta seperti:
- Nama lengkap
- Nama panggilan
- Tanggal lahir
- Jenis kelamin
- Nomor HP
- Email
- Agency
- Tinggi badan
- Berat badan

Setiap peserta akan mendapatkan ID unik secara otomatis.

---

### 📋 Melihat Seluruh Data Peserta
Menampilkan seluruh peserta yang telah terdaftar dalam format tabel yang rapi dan mudah dibaca.

Informasi BMI peserta juga langsung ditampilkan tanpa perlu perhitungan manual.

---

### 🔍 Mencari Peserta
Tidak perlu scroll satu per satu.

Program menyediakan pencarian berdasarkan:
- Nama peserta
- ID peserta

Sehingga data yang dicari dapat ditemukan dengan lebih cepat.

---

### ✏️ Mengubah Data Peserta
Jika terdapat kesalahan data atau peserta ingin memperbarui informasinya, data dapat diedit kapan saja tanpa perlu membuat data baru.

---

### 🗑️ Menghapus Data Peserta
Peserta yang tidak lagi diperlukan dapat dihapus langsung dari sistem dengan mudah.

---

### 📊 Mengurutkan Data Peserta
Data dapat diurutkan sesuai kebutuhan panitia, antara lain berdasarkan:

- ID
- Nama
- Tinggi badan
- BMI

Tersedia pilihan urutan naik (ascending) maupun turun (descending).

---

### ⚖️ Analisis BMI Otomatis
Program akan menghitung BMI setiap peserta dan mengelompokkannya ke dalam kategori:

- Underweight
- Normal
- Overweight
- Obesitas

Fitur ini membantu memberikan gambaran kondisi fisik peserta secara cepat.

---

### 📈 Statistik Peserta
Selain BMI individu, program juga dapat menampilkan statistik keseluruhan peserta, seperti:

- Jumlah peserta pada setiap kategori BMI
- Rata-rata tinggi badan
- Rata-rata berat badan

---

### 💾 Penyimpanan Data Permanen
Semua data peserta akan disimpan ke file secara otomatis.

Saat program dijalankan kembali, data yang tersimpan sebelumnya akan langsung dimuat sehingga tidak perlu menginput ulang.

---

## 🎯 Tujuan Project

Project ini dibuat untuk menerapkan konsep-konsep dasar pemrograman C++ dalam sebuah studi kasus yang dekat dengan dunia nyata, seperti:

- Pengelolaan data peserta
- Pencarian data
- Pengurutan data
- Pengolahan statistik
- Penyimpanan data ke file

Dengan pendekatan tersebut, program tidak hanya berfungsi sebagai tugas akademik, tetapi juga menyerupai sistem administrasi sederhana yang benar-benar dapat digunakan.

---

## 🚀 Cara Menjalankan

1. Compile program menggunakan compiler C++.
2. Jalankan file hasil compile.
3. Pilih menu yang tersedia.
4. Kelola data peserta sesuai kebutuhan.

---

## 📝 Catatan

- Maksimal data yang dapat disimpan adalah 100 peserta.
- Data akan tersimpan secara lokal pada file penyimpanan.
- Program berjalan melalui terminal/command prompt.
- Tidak memerlukan database atau instalasi tambahan.
